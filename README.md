# SHOP

### Setup

##### Prerequisites

[Download and install Node.js.](https://nodejs.org/en/download/)

Install [polymer-cli](https://github.com/Polymer/polymer-cli):

    $ npm install -g polymer-cli

Install [bower](https://bower.io/):

    $ npm install -g bower

##### Setup

    $ npm install

### Start the development server

    $ npm start
